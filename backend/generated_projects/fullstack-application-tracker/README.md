# fullstack-application-tracker

A full-stack dashboard for tracking applications, statuses, analytics, and editable application materials.

## Why this project exists
This project is designed to strengthen a resume for **Full Stack Engineer** roles.

## Core features
- Demonstrates React
- Demonstrates TypeScript
- Demonstrates Python
- Demonstrates FastAPI
- Demonstrates PostgreSQL
- Demonstrates Docker

## Run locally
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn src.main:app --reload
```
